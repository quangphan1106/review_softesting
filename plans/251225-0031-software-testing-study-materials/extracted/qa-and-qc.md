## EN.QA and QC.pdf

# QA Vs QC

- QA – Quality Assurance
- QC – Quality Control

# ISO 9000 Definitions

- [ ] QC:
    - The operational techniques and activities that are used to fulfill requirements for quality
- [ ] QA:
    - All those planned and systematic activities implemented to provide adequate confidence that an entity will fulfill requirements for quality

# Typical QC steps

## Problem Management

The typical QC steps form a flow as follows:

1.  Problem Identification
2.  Problem Analysis
3.  Problem correction (from Problem Analysis)
4.  Feedback to QA (from Problem Analysis)
5.  O/P to QA (from Feedback to QA)

# Typical QA steps

The typical QA steps form a flow as follows:

1.  I/P from QC
2.  Data Gathering
3.  Problem Trend Analysis
4.  Process Identification
5.  Process Analysis
6.  Process Improvement
